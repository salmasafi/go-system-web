## 🔧 التوصيات الفنية

### 1. استخدام نظام الترجمة بشكل كامل
بدلاً من النصوص المكتوبة مباشرة (hardcoded)، يُنصح بـ:

```dart
// ❌ سيء
Text('Add Product')

// ✅ جيد
Text(LocaleKeys.add_product.tr())
```

### 2. إضافة المفاتيح المفقودة إلى ملف الترجمة
أضف جميع النصوص المفقودة إلى:
- `assets/translations/ar.json`
- `assets/translations/en.json`

### 3. توليد ملف المفاتيح
بعد تحديث ملفات JSON، قم بتشغيل:
```bash
flutter pub run easy_localization:generate -S assets/translations -f keys -o locale_keys.g.dart
```

### 4. التحقق من الترجمات
استخدم أداة للتحقق من:
- المفاتيح المفقودة
- الترجمات غير المستخدمة
- التنسيق الصحيح
